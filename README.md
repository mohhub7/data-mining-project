# Data Mining Project

## Algorithms Implemented
- **Apriori** - Association rule mining (market basket analysis)
- **K-Means Clustering** - Unsupervised learning to group similar data
- **Decision Tree** - Classification using entropy and information gain

## Files
| File | Description |
|------|-------------|
| `apriori.py` | Finds frequent itemsets and association rules |
| `clustering.py` | K-Means clustering with elbow method |
| `decision_tree.py` | Builds decision tree for classification |
| `clustering_result.png` | Visualization of clusters |
| `decision_tree.png` | Visualization of the decision tree |

## How to Run
```bash
pip install -r requirements.txt
python apriori.py
python clustering.py
python decision_tree.py
